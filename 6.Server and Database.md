# Server
Three types of server are there :-
## 1. Hardware Server :-
- This is actual machine with RAM, ROM which is 24/7 running to serve the clients.
## 2. Software Server :-
- This is the software inside the harware server which is used to listen request, manage connection, respond to requests, etc.
## 3. Application server :-
- Application running on software to listen to responses and open sockets to connect with other servers and bundle up the data in the required format and send it back.
- The one we make through nodejs is application server 
### There are many formats of responses from the server like:-
1. HTTP (Hyper Text Transfer Protocol) :- We make this with nodejs.
2. SMTP (Simple Mail Transfer Protocol)
3. FTP (File Transfer Protocol)
> - When we type domain name it maps to the ip address of the server machine and browser fetches it through DNS (domain name server).
> - There can be many applications running on the same software server so for the specific application we need port number and we also define by default port so we do not have to write port number when we enter domain name.
> - Now ip address and and port number gives the application with which socket has to be open but for the specific data we need to write the path and after requiring path request is fully sent.
> - Writing the path gives the api back also we can define default data in the api for the empty path which loads the website.

# Database
- The organized collection of related data, designed to allow efficient access, retrieval and modification of data is called database.
- A software which allows users to define, create, maintain and control access to database is called DBMS.
- Database system is complete environment consisting of the database, the DBMS, application programs and the users that work together to store and operate on data.
> It is possible that database and web server are hosted on different server machines.
